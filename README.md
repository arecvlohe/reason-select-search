## Shortcuts

Here is a list of the keyboard shortcuts:

| Key        | Action                                          |
| ---------- | ----------------------------------------------- |
| [          | Open dropdown                                   |
| ]          | Close dropdown                                  |
| s          | Focus search bar                                |
| down arrow | Navigate dropdown list going down               |
| up arrow   | Navigate dropdown list going up                 |
| enter      | Select the focused element in the dropdown list |
